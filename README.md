# HR Analytics End-to-End Project

## Project Overview

This project analyzes employee data to understand workforce trends and employee attrition.
The goal of the project is to identify patterns in employee behavior, department-wise attrition, and other HR metrics that can help organizations improve employee retention and decision-making.

## Tools & Technologies Used

* **Excel** – Data cleaning and initial analysis
* **SQL** – Data exploration and querying key HR metrics
* **Power BI** – Interactive dashboard and KPI visualization
* **Tableau** – Data visualization and analytical dashboard

## Dataset

The dataset contains employee information such as:

* Employee ID
* Department
* Job Role
* Age
* Salary
* Job Satisfaction
* Attrition Status
* Years at Company

This data is used to analyze workforce distribution and identify factors contributing to employee attrition.

## Key KPIs Analyzed

* Total Employees
* Attrition Count
* Attrition Rate
* Average Age of Employees
* Department-wise Attrition
* Job Satisfaction Levels

## Project Workflow

1. Data cleaning and preprocessing using **Excel**
2. Data exploration using **SQL queries**
3. Data visualization and KPI creation using **Power BI**
4. Interactive dashboard creation using **Tableau**

## Key Insights

* Certain departments show higher attrition compared to others.
* Employees with lower job satisfaction tend to leave more frequently.
* Attrition is higher among employees with fewer years at the company.
